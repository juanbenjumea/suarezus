                    COMO INSTALAR UNA NUEVA VERSION
                       Version 1, Julio 2016
					   

Una vez descargado el proyecto de Github (El repositorio debio ser instalado en www/suarez/us)

1. Copiar archivos de la version de wordpress deseada en proyecto
2. Base de datos.
	- Crear usuario que aparece en wp-config
	- Crear esquema "suarez_us"
	- Dar permisos al usuario sobre este esquema y ejecutar el script que encuentra en "wp-install/db/suarez_us.sql"
3. Extraer plugins que se encuentran en "wp-install/plugins/" en la carpeta "wp-content/plugins"


That's all, ENJOY IT
